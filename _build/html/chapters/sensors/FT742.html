FT742 Wind sensor#

The FT7 Series of ultrasonic wind sensors deliver highly accurate and consistent wind speed and direction data from compact sensors. With no moving parts, built-in heaters, and resistance to a wide range of environmental factors, the FT7 series sensors are maintenance-free and will last for years, even in the harshest of climates. More information.

$FT742.ENA#

Description: Enable the DCPS
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 1
Example: $FT742.ON 1

Enables the DCPS, $FT742.ENA 1 to turn it on, and $FT742.ENA 0 to turn it off. This can be useful to conserve battery.

$FT742.RNO#

Description: Run order
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 9
Example: $FT742.RNO 1

Sets the Run order for the sensor. See Run order

$FT742.RNC#

Description: Run counter
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: -
Example: $FT742.RNC 1

Sets the Run counter for the sensor. See Run counter

$FT742.GPS#

Description: Enable the GPS
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 2
Example: $FT742.GPS 1

Enable the logging the start and end positions of the sensor acquisition.

$FT742.DEV#

Description: Set the magnetic declination
No. of parameters: 1
Unit: degrees
Parameter type: integer
Minimum parameter value: -180
Maximum parameter value: 180
Example: $FT742.DEV -21

The magnetic deviation used when transmitting surface currents. This will need to be set depending on location, Baltic Sea is ~+4°, while the SO can be ~-20°. Find out more about magnetic declination here

$FT742.ONT#

Description: Measurement time
No. of parameters: 1
Unit: seconds
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 1800
Example: $FT742.ONT 20

The number of seconds the FT742 sensor is set to acquire data.

$FT742.LOG#

Description: Log output
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 1
Example: $FT742.LOG 1

Log sensor output to disk.

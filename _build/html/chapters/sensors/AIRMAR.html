Airmar 200WX#

The Airmar 200WX is a Weatherstation containing various sensors. Documentation can be found on www.airmar.com

The sensor is configured using the WeatherCaster software from Airmar. The following sentences should be enabled:

  • $WIMDA – windspeed, direction

  • $WIMWV – windspeed if no gps fix

  • $HCHDG - heading

$AIRRP#

Description: Run order
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 1
Maximum parameter value: 9
Example: $AIRRP 1

Sets the Run order for the sensor. See Run order

$AIRRC#

Description: Run counter
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: -
Example: $AIRRC 1

Sets the Run counter for the sensor. See Run counter

$AIREN#

Description: Enable the Airmar
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 1
Example: $AIREN 1

Enables the Airmar, $AIREN 1 to turn it on, and $AIREN 0 to turn it off. This can be useful to conserve battery.

$AIRTO#

Description: Measurement time
No. of parameters: 1
Unit: seconds
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 3600
Example: $AIRTO 60

The number of seconds the Airmar is set to acquire data. With $AIRAO 1 it is fine to have 60 seconds, but with $AIRAO 0 we would require at least 15 minutes to achieve a GPS fix.

$AIRAO#

Description: Set the Airmar Always On
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 1
Example: $AIRAO 1

The Airmar can be configured to always be on, which allows it to get a GPS fix. The fix is required for accurate measurements of wind direction, and is sometimes difficult to get. This is why we usually leave the Airmar on between measurements. The power consumption increases massively, and should only be done in good light conditions, so that the battery are allowed to fully recharge during the day. Send $AIRAO 1 to activate this function.

$AIRLG#

Description: Log output
No. of parameters: 1
Unit: -
Parameter type: integer
Minimum parameter value: 0
Maximum parameter value: 1
Example: $AIRLG 1

Log sensor output to disk.

Note

If the sensor is not powered on long enough the AirmarAirFix will be 0. This means the sensor has not gained a GPS fix and the data transmitted will be of a lower quality.

Mounting#

Before attaching the sensor, it is important to lubricate around the connector with marine silicone to ensure that water does not enter the pins when the sensor is mounted. Hand tighten the screw. Align the sensor with the alignment lines.
